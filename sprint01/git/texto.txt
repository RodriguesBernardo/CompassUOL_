Comandos Importantes do Git

git pull                    - Recebe atualizações do repositorio
git push                    - Envia as atualizações para o repositorio
git commit                  - Prepara para o push
git remote -v               - verifica as origens do fetch e push
gite remoterm origin        - retira a origem do servidor

git mv                      - Movendo Arquivos
git checkout                - CUIDADO(faz o reset no arquivo conforme tem na main)
.gitignore                  - Cria pasta com informações de caminhos a serem ignorados pelo git
git reset                   - Resetar mudanças feitas, todas alterações commitadas e também pendentes serão excluidas
git clone $urldogit         - Clona arquivos do repositorio

git branch                  - Mostra todas as branchs disponiveis no projeto
git branch $nomeDaBranch    - Cria nova branch 
git branch -d $nomeDaBranch - Deleta a branch (Não é comum)
bit checkout $nomeDaBranch  - Muda de branch
git merge $nomeDaBranch     - Junta duas branchs
git tag -a $nome da tag -m  - Cria nova Tag 
git diff                    - Verifica diferenças 
git gc                      - Otimiza o repositorio
git fsck                    - Verifica possiveis corrupção em arquivos (fazer uma vez por mês) 
git archive --format zip --output $nomedoArquivo.zip master    -Transforma o repositorio em um arquivo zip


        Boas mensagemns de commit
    - Separar assunto do corpo da mensagem;
    - Assunto com no máximo 50 caracteres;
    - Assunto com letra inicial maiuscula;
    - Explicar o porque e como do commit, e não como o código foi escrito;
